# ✅ Checklist MCMV

Aplicativo web para corretores de imóveis gerenciarem e acompanharem a documentação necessária para financiamentos pelo programa 
**Minha Casa Minha Vida (MCMV)**.

## 🚀 Acesse o app

👉 [cheerful-dragon-620c59.netlify.app](https://cheerful-dragon-620c59.netlify.app)

## 🛠️ Tecnologias utilizadas

- [Expo](https://expo.dev/) + React Native Web
- Deploy via [Netlify](https://netlify.com)

## 💡 Funcionalidades

- Checklist interativo de documentos exigidos pelo MCMV
- Interface responsiva para uso no celular e desktop
- Acesso rápido para corretores em campo

## 📦 Como rodar localmente

```bash
npm install
npx expo start
```
